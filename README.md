# Terraform-AWS-Projects

Welcome! This repository contains my collection of infrastructure projects deployed on **Amazon Web Services (AWS)** using **Terraform**. 

## 🚀 Featured Projects

* **Three-Tier Web Architecture**: 
    Automated deployment of a scalable and highly available web application environment, including:
    * **Public/Private Subnets** for network isolation.
    * **Auto Scaling Groups** and **Load Balancers**.
    * **RDS Instances** for managed database services.

## 🛠️ Key Technologies
* **Infrastructure as Code (IaC):** Terraform
* **Cloud Provider:** AWS (VPC, EC2, RDS, S3, IAM)
* **Architecture:** Multi-AZ deployments for high availability.

## 📂 Project Structure
Each project is housed in its own directory with dedicated `.tf` configuration files.
